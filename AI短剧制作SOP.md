# AI 短剧制作 SOP（可复用生产线 · 执行+标准合一版）

**版本**：v3.0 ｜ **归属**：阶段 1 交付物③（内容获客生产线）
**用途**：古风餐厅经营 + 合成 小游戏的内容获客执行流程与引擎标准
**定位**：AI 短剧不是"营销素材"，是产品的延伸、是 unit economics 的承重墙。本 SOP 由「AI 短剧制作流程.md」与「AI 短剧内容引擎SOP.md」合并而成，按**换皮出海零成本复用**标准搭建——主题、模板、流水线、精修红线全部抽象成可迁移资产，换题材只换皮、不换骨架。

> 硬指标以 `项目计划书.md` §6 验证指标体系为权威源。本文件既讲"**做一条短剧要走的 8 个步骤 + 每步对应的工具**"，也沉淀可复用的引擎资产（钩子库、脚本模板、精修红线、埋点标准、换皮清单）。

---

## 0. 流程总览（8 步流水线，两种产出）

```
① 剧本选题(主题库+钩子库) → ② 剧情脚本(分镜) → ③ 角色&场景设定(GPT-Image-2)
   → ④ 静态分镜生成(GPT-Image-2) → ⑤ 动态视频生成(SeeDance 2) → ⑥ 剪辑配音后期
   → ⑦ 成片发布 → ⑧ 数据反馈(三色灯)
                                              │
                                              └─ 游戏侧承接(角色客串/剧情钩子) → 回流
```

- **对外产出**：抖音 / 视频号 / 小红书 / TikTok Minis 的剧情向短剧（每周 2–3 条）。
- **对内产出**：可被游戏复用的剧情钩子（驱动次日回流）、角色人设（游戏内 NPC）。
- **关键纪律**：每条短剧在**分镜阶段就埋好游戏钩子**，不是成片后硬贴链接。

## 1. 剧本选题：主题储备 + 钩子库（引擎资产·钩子库/人设库）

**目标**：前期储备多套主题，建立"通用钩子 + 特定钩子"双层钩子库，产出本条「选题卡」。

**对应工具**
- **主题库 / 钩子库管理**：飞书多维表格（或 Notion / Airtable / Excel）建「主题×钩子」双表，字段含：母题、钩子类型（通用/特定）、目标 CTR、映射游戏节点、已用次数、状态（绿/黄/红）。
- **钩子扩写辅助**：对话式大模型（GPT-5 / Claude / Kimi）按母题批量生成钩子候选，人审后入库。

**主题储备纪律**
- 主投主题：**大女主古风餐厅逆袭**（设定见下方「创作圣经」）。
- **常备 ≥ 2 套备用主题**（不投时也要有）：其一即换皮出海版「中世纪女厨酒馆逆袭」（豆豆眼餐馆），其二为后续题材试验田。红灯触发时 24h 内切换，不空档。

**通用钩子 vs 特定钩子**
- **通用钩子**（跨主题复用，情绪底层）：打脸爽点、逆袭高光、公平反击、独立清醒、悬念卡点。
- **特定钩子**（按母题定制，绑定游戏节点）：见下表「五大母题 × 钩子 × 游戏解锁」。

**大女主剧情设定（创作圣经 · 本主题统一世界观）**

> 全剧以**苏望月**为唯一核心驱动者：她搞事业、她手撕反派、她独美——**不靠男主救、不恋爱脑**。所有母题、角色、分镜都服务于"女主逆袭 + 古风餐厅经营"。换皮出海时仅替换文化外壳（中世纪女厨 / 酒馆主），大女主内核 100% 复用。

- **世界观**：融合唐宋明的架空古风，以"餐厅里的人和事"为叙事容器。名厨之女苏望月被继母与渣未婚夫陷害、逐出家族、夺走祖传食坊 → 接手濒临倒闭的破店（望月食坊前身）→ 凭厨艺 + 经营脑重振 → 手撕仇家、做成第一食坊。情绪核心：**打脸爽感 + 事业高光 + 清醒独立**。
- **女主弧光（五拍）**：① 被逐立誓（破店雨夜）② 一道菜立威（失传菜惊艳全城）③ 手撕反派（继母/渣男/同行砸场反杀）④ 经营破局（资金链/打压危机）⑤ 独美登顶（破店→第一食坊，不婚独美）。

| 母题 | 大女主题材（特定钩子） | 游戏内解锁 |
|------|----------------------|-----------|
| ① 厨艺打脸 | 用一道失传菜让渣男/恶毒女配/势利眼闭嘴 | 合成链解锁 |
| ② 身世逆袭 | 被逐真相、祖传食谱秘方、归来 | 主线剧情节点 |
| ③ 经营危机 | 退租/资金断/同行打压 → 女主经营脑化解 | 经营数值事件 |
| ④ 合成彩蛋 | 食谱第三页/失传配方 → 游戏合成玩法 | 合成玩法入口 |
| ⑤ 高光节日 | 花朝/中秋宴 → 女主高光、手撕名场面 | 节日限定活动 |

- **反派 ensemble（衬托大女主，非救赎者）**：赵氏（继母·恶毒女配，夺产散谣）、陆三郎（退婚渣男，嫌贫爱富）、金满楼掌柜（同行打压，势利眼抢食材砸场）。
- **伙伴（成长线，不抢女主光）**：小满（学徒厨娘，头号迷妹→游戏新手陪伴）、老顾（退休名厨，授秘技→合成链解锁者，师徒非恋爱）、食魂（古旧食谱精灵→合成玩法 mascot）。

**选题卡模板**（每条必填）：本次主题、钩子类型（通用/特定）、对应母题、目标 CTR、目标完播率基线、映射游戏节点。

## 2. 剧情脚本：分镜细化（明确工具 · 引擎资产·脚本模板）

**目标**：套模板产出 ≤ 60 秒分镜脚本，前 3 秒有钩子、剧末 CTA 清晰。

**对应工具**
- **脚本撰写**：对话式大模型（**GPT-5**，备选 Claude / Kimi）+ 结构化分镜模板（人维护）。
- **喂模型的固定输入**：创作圣经（§1）+ 选定钩子 + 分镜表格式 + 埋钩栏要求。Prompt 要点：「你是古风大女主短剧编剧，严格遵循苏望月主导、不恋爱脑红线，按下方分镜表格式输出 N 镜，钩子前置，剧末给游戏 CTA」。

**脚本模板（分镜标准格式 · 大女主开场示范）**

| 镜号 | 景别 | 画面描述 | 台词 / 旁白 | 时长 | 音效 / BGM | 游戏钩子 |
|------|------|---------|------------|------|-----------|---------|
| 1 | 全景 | 破败食坊，雨夜，望月立门前 | 旁白："被赶出苏家那夜，他们说我这辈子开不了店" | 3s | 雨声+古筝 | 无 |
| 2 | 特写 | 退婚书掷地，陆三郎冷笑离场 | "陆三郎说，落魄女不配当他妇" | 4s | 纸声 | 无 |
| 3 | 中景 | 望月擦灶台，眼神凌厉 | "那我就开给你们看——望月食坊，今晚开张" | 5s | 点火声 | 女主立誓=主线起点 |
| 4 | 特写 | 一锅失传"望月羹"起锅，金光 | "这道菜，苏家失传十年" | 5s | 滋滋声 | 食谱=合成入口 |
| 5 | 中景 | 赵氏带人砸场，望月举勺挡 | "我的店，轮不到你撒野" | 6s | 碎瓷声 | 角色客串→进游戏 |
| … | … | … | … | … | … | … |
| N | 卡点 | 望月背影，店招牌亮 | "想知道我把破店做成第一食坊？来游戏里开" | 2s | 鼓点骤停 | CTA：评论区/简介链 |

**结构铁律**：钩子前置（镜 1–3，被逐/悔婚/立誓三连击）、情绪中段（一道菜立威+手撕）、悬念卡点收尾（剧末=游戏入口）。
**大女主红线**：女主必须是行动发起者；反派制造压迫、女主反杀；禁止"等男主来救"。
**埋钩纪律**：游戏钩子栏在分镜阶段就必须填，禁止成片后硬贴链接。

## 3. 角色 & 场景设定（GPT-Image-2 · 引擎资产·人设库）

**目标**：产出角色卡 + 场景概念图，锁定视觉一致性，供后续分镜复用。

**对应工具**
- **生图模型**：**GPT-Image-2**（角色卡、场景概念图、道具/服饰设定）。
- **一致性保障**：固定 seed + 参考图（.seed / 角色参考图），统一服饰/发式/器物基调；建立「角色视觉档案」表（每人 1 张主卡 + 表情/姿态变体）。

**大女主阵容人设表（生图依据）**

| 角色 | 定位 | 性格钩子 | 游戏内复用 | 生图提示要点 |
|------|------|---------|-----------|------------|
| **苏望月** | **大女主·食坊主理** | 清醒独立、事业脑、毒舌护短、厨艺封神 | 玩家化身/主线引导 NPC | 唐宋明古风、利落束发、灶台前凌厉眼神 |
| 赵氏 | 反派·继母（恶毒女配） | 夺产散谣、笑里藏刀 | 阶段剧情反派节点 | 华服、假笑、权贵气场 |
| 陆三郎 | 反派·退婚渣男 | 嫌贫爱富、悔婚打脸对象 | 支线打脸事件 | 纨绔锦衣、倨傲 |
| 金满楼掌柜 | 反派·同行打压 | 势利眼、抢食材砸场 | 经营危机事件 | 富态、市侩 |
| 小满 | 伙伴·学徒厨娘 | 头号迷妹、笨拙可爱、成长线 | 新手教学陪伴 | 圆脸、围裙、雀跃 |
| 老顾 | 伙伴·退休名厨 | 认可女主、授秘技（师徒非恋爱） | 合成链解锁者 | 白发、沉稳、灶王气场 |
| 食魂 | 古旧食谱精灵 | 合成彩蛋载体 | 合成玩法 mascot | 半透明古卷精灵、发光 |

**场景概念图**：望月食坊（破店→修缮后）、苏家大宅、市井街巷、花朝/中秋宴席——均按唐宋明架空基调出图，存入场景库供分镜调用。

## 4. 静态分镜生成（GPT-Image-2）

**目标**：按 §2 分镜表逐镜生成关键帧，作为动态视频底图。

**对应工具**
- **生图模型**：**GPT-Image-2**（关键帧/角色卡复用 §3 视觉档案，保证同镜人物一致）。

**操作纪律**
- 逐镜调用：prompt 含景别 + 动作 + 古风质感 + 角色参考图；同一角色跨镜必须引用 §3 视觉档案的 seed/参考图，杜绝换镜变脸。
- **非卡通廉价感**：色调、服饰、器物符合唐宋明架空基调；拒绝塑料感高光、扁平色块。
- 产出：每条短剧一套关键帧（PNG），命名按 `镜号_描述`，进动态视频环节。

---

## 5. 动态视频生成（SeeDance 2）

**目标**：将静态关键帧转化为动态镜头，产出粗剪视频。

**对应工具**
- **视频生成模型**：**SeeDance 2**（图生视频 / 文生视频）。
- **口型同步**：对白镜头用 SeeDance 2 口型对齐能力，动作自然不抽搐。

**操作纪律**
- 逐镜图生视频，控制镜头运动幅度（古风稳重，避免乱飞）；对白镜绑定 §6 配音音轨做口型对齐。
- 产出粗剪视频（按分镜顺序排列），流转剪辑环节。

---

## 6. 剪辑配音后期（引擎资产·精修红线）

**目标**：过精修红线，产出可发布成片。

**对应工具**
- **剪辑/配音/字幕**：剪映专业版 / CapCut（含 AI 配音、智能字幕、一键调色、古风模板）。
- **配音音色**：古风 TTS / AI 配音，清晰无机械感，产出音轨并与 §5 口型对齐。

**精修红线（古风审美洁癖 checklist）**——任一项不达标 → **打回重做，不成片发布**：
- **画面**：无 AI 脸崩/手指异常/肢体扭曲；古风质感（非卡通廉价）；色调统一不跳帧。
- **节奏**：前 3 秒有钩子；总时长 ≤ 60s；完播友好（每 8–10s 一个情绪点）。
- **声音**：配音清晰无机械感；BGM 不抢戏、契合古风。
- **转化**：剧末有明确游戏钩子；CTA 清晰（"同款餐厅，来游戏里开"）。

---

## 7. 成片发布（引擎资产·游戏转化埋点）

**目标**：剧游联动分发，挂链承接回流。

**对应工具**
- **发布平台**：抖音创作者中心 / 微信视频号 / 小红书 / TikTok Minis（出海验证后）。
- **埋钩挂链位置三处一致**：评论区置顶 + 视频简介 + 剧内口播。
- **分镜即埋钩**：剧中望月手撕反派时说"这店我开定了，配方在我手里" → 自然引导进游戏（女主主导，非被动客串）。
- **剧末卡点**：悬念停在游戏可继续处（"食谱第三页写了什么？来望月食坊看"）。
- **角色闭环**：短剧角色在游戏内作为 NPC 出场，观众进游戏"见到熟人"→ 回流强化。
- **周更节奏**：每周 2–3 条（周一/三/五发，周末复盘）。
- **硬指标**：短剧→游 **CTR ≥ 3%**（低于即触发 §8 三色灯黄灯，对齐 `项目计划书.md` §10.4）。

## 8. 数据反馈：三色灯机制（引擎资产·指标看板）

**目标**：用三色灯快速判定主题剧本生死，跑不出立即换，闭环反哺钩子库。

**对应工具**
- **数据采集**：平台原生后台（抖音/视频号/小红书/TikTok 数据看板）。
- **看板搭建**：飞书多维表格（CTR / 完播率 / 涨粉·自然量占比 / 进游注册数 四列 + 条件格式自动三色灯）。

**每条必盯指标**（对齐 `项目计划书.md` §7.2 看板）：完播率（内容质量）、涨粉/自然量占比（引流能力）、**短剧→游 CTR（≥3% 硬指标）**、进游注册数（转化能力）。

**三色灯判定**

| 灯 | 判定阈值 | 动作 |
|----|---------|------|
| 🟢 绿灯 | CTR ≥ 3% 且 完播率 ≥ 基线（如 35%） | 该主题/钩子进"可批量"队列，提升周更占比 |
| 🟡 黄灯 | CTR 1.5%–3% 或 完播率低于基线 | 保留但限 1–2 条试验，换钩子角度/前3秒/封面/CTA，不进批量 |
| 🔴 红灯 | CTR < 1.5% 连续 2 条（同主题/钩子）；或单条 CTR < 1% | **立即砍掉该主题剧本**，从 §1 储备库切换下一套主题 |

**快速更换纪律**
- 储备库必须常备 ≥ 2 套备用主题 + 各自钩子（§1 纪律），红灯触发后 **24h 内上线替换主题第 1 条**，避免内容空档。
- "一个主题剧本跑不出就快速换"——不恋战、不补拍、不解释，直接切下一套。
- **周复盘**：绿灯钩子 → 反哺钩子库（提高权重、批量复用）；红灯主题 → 标记淘汰、记录失败原因供备用主题避坑。
