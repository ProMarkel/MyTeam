## 阅读顺序建议
1. 先读 `项目计划书.md` 建立全局（商业论证 + 立项决策一体）
2. 研发按 `MVP设计文档.md` 落地设计
3. 内容按 `AI短剧制作SOP.md` + `AI短剧-第1条脚本.md` 起步（也可直接在 `index.html` 的「AI短剧制作SOP」段看完整生产线 + 示例①）
4. 上线后用 `验证看板.md` 盯数
5. 对外展示用 `index.html`

> 凡涉及量化门槛（次留 / CPA / LTV / 红黄灯），一律以 `项目计划书.md` §6 及其延伸章节为权威源，避免多文档口径分歧。
