# 写作关键规则（区别于普通简历）

1. **全部量化，拒绝空话** —— 流水、日活、付费率、团队规模、上线平台均取自真实履历，不写"负责数值 / 做活动"这类空描述。
2. **只写和当前赛道匹配项目** —— 国风 / 女性向 / 模拟经营 / 小游戏相关经历重点保留；早期非相关品类在全文履历中保留、此处剔除。
3. **每段结尾绑定本次项目价值** —— 每位成员明确写清"本次项目负责"与"核心优势如何降低本项目风险"。
4. **篇幅极精简** —— 每人 4–6 行，无爱好、无基础软件技能罗列、无校园经历、无短期实习。
5. **突出团队互补** —— 策划 / 数值 / 美术 / 前端等核心能力闭环，核心成员已在同一女性向模拟经营产品中长期磨合（前端由主程 + 开发双岗互补）。发行与版号由制作人统筹，不单独设岗。
